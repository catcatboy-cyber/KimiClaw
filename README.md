# 🦞 KimiClaw 小龙虾 - 桌面悬浮电子宠物

一款可爱的Android桌面悬浮宠物应用，小龙虾会在你的手机屏幕上爬来爬去，陪你聊天，还能帮你监控社交App消息！

## ✨ 功能特点

### 🦞 桌面悬浮宠物
- 小龙虾在手机桌面上自由爬动
- 不影响正常操作（可穿透点击）
- 可拖动到任意位置
- 点击互动（会跳、会说话）

### 🍤 喂养系统
- 饥饿度会随时间下降
- 点击"喂食"按钮喂养
- 饥饿时小龙虾会提醒

### 💬 AI对话
- 点击小龙虾或主界面"对话"按钮
- 和小龙虾聊天互动
- 智能回复（支持中文）

### 📱 消息监控
- 监控微信、QQ等社交App消息
- 指定联系人消息提醒
- 小龙虾会跳动提醒你

## 📋 系统要求

- Android 6.0 (API 24) 及以上
- 需要悬浮窗权限
- 需要通知监听权限（用于消息监控）

## 🔧 安装方法

### 方法一：直接安装APK
1. 下载 `KimiClaw-v1.0.apk`
2. 允许安装未知来源应用
3. 安装并打开应用

### 方法二：从源码构建

#### 环境要求
- Android Studio Hedgehog (2023.1.1) 或更高版本
- JDK 17
- Android SDK 34

#### 构建步骤

1. 克隆或下载源码
```bash
cd KimiClawPet
```

2. 使用Android Studio打开项目
   - 选择 `File` → `Open` → 选择项目文件夹
   - 等待Gradle同步完成

3. 构建APK
   - 选择 `Build` → `Build Bundle(s) / APK(s)` → `Build APK(s)`
   - 或 `./gradlew assembleDebug`

4. 安装APK
   - 连接手机，开启USB调试
   - 点击运行按钮，或手动安装生成的APK

## 🚀 使用说明

### 首次启动
1. 打开应用，点击"启动小龙虾"
2. 授予**悬浮窗权限**（系统会跳转设置页面）
3. 授予**通知监听权限**（用于消息监控）

### 基础操作
- **启动/停止**：主界面按钮控制
- **移动**：长按并拖动小龙虾
- **互动**：点击小龙虾，它会跳并说话
- **喂食**：点击"喂食"按钮增加饥饿度

### 消息监控设置
1. 点击"消息监控设置"
2. 添加要监控的联系人名称
3. 选择要监控的App（微信、QQ等）
4. 保存设置

当指定联系人发来消息时，小龙虾会跳动并显示提醒！

## 🎮 小龙虾行为

| 状态 | 表现 |
|------|------|
| 开心 | 正常爬动，偶尔说话 |
| 饥饿 | 显示🍤图标，说"我饿了" |
| 收到消息 | 跳动，显示发送者名称 |

| 被喂食 | 变大动画，说"好吃！" |

## 🐛 常见问题

### Q: 小龙虾不显示？
A: 请确保已授予悬浮窗权限。去设置 → 应用 → KimiClaw → 显示悬浮窗

### Q: 消息监控不工作？
A: 需要开启通知监听权限。去设置 → 通知 → 通知使用权 → 开启KimiClaw

### Q: 耗电吗？
A: 优化过的前台服务，耗电量很低。可以在不需要时点击"停止小龙虾"

### Q: 支持哪些社交App？
A: 目前支持微信、QQ、微博、钉钉、WhatsApp、Telegram
