## 详细描述（应用商店完整版）

### 🦞 遇见你的专属桌面宠物

MelodyClaw 是一款充满趣味的Android桌面悬浮宠物应用。一只可爱的小龙虾会在你的手机屏幕上自由爬动，陪你聊天互动，还能帮你监控重要联系人的消息！无论是工作还是娱乐，小龙虾都会陪伴在你身边。

---

### ✨ 核心功能

**🦞 桌面悬浮宠物**
• 小龙虾在屏幕上自由爬动，动作生动可爱
• 不影响其他应用操作（支持点击穿透）
• 可随意拖动到任意位置
• 点击互动，小龙虾会跳跃并说话
• 多种表情状态：开心、饥饿、兴奋、难过

**🍤 养成系统**
• 饥饿度随时间自然下降
• 及时喂食让小龙虾保持活力
• 饥饿时会主动提醒你
• 心情值影响小龙虾的行为表现
• 等级系统，陪伴越久等级越高

**💬 AI智能对话**
• 点击小龙虾即可开启对话
• 支持自然语言聊天
• 可爱俏皮的回复风格
• 使用智谱GLM AI技术
• 可配置自己的API Key

**📱 消息智能监控**
• 监控微信、QQ、钉钉等社交应用
• 指定重要联系人消息提醒
• 收到消息时小龙虾会跳动提醒
• 锁屏状态也能及时通知
• 消息内容仅本地处理，绝不上传

**🔔 贴心提醒功能**
• 重要消息不错过
• 可自定义监控的联系人列表
• 支持多个社交应用同时监控
• 消息弹窗显示，一键跳转回复

### 🔐 隐私与安全

我们非常重视您的隐私安全：

✅ **本地处理**：所有消息内容仅在本地设备处理，绝不上传服务器
✅ **数据加密**：敏感信息加密存储
✅ **代码混淆**：应用代码已混淆，防止逆向工程
✅ **日志脱敏**：日志中的敏感信息已脱敏处理
✅ **签名验证**：自动更新时验证APK签名，防止恶意篡改
✅ **透明政策**：完整的隐私政策说明

### 🚀 快速上手

1. **首次启动**：打开应用，点击"启动小龙虾"

3. **开始互动**：点击小龙虾聊天，点击喂食按钮喂养
4. **设置监控**：在设置中添加要监控的联系人

### 🐛 常见问题

**Q: 小龙虾不显示？**
A: 请确保已授予悬浮窗权限。路径：设置 → 应用 → MelodyClaw → 显示悬浮窗

**Q: 消息监控不工作？**
A: 需要开启通知监听权限。路径：设置 → 通知 → 通知使用权 → 开启MelodyClaw

**Q: 耗电吗？**
A: 经过优化的前台服务，耗电量很低。不需要时可点击"停止小龙虾"

**Q: 支持哪些社交App？**
A: 目前支持微信、QQ、微博、钉钉、WhatsApp、Telegram等主流应用

**Q: AI对话需要付费吗？**
A: 需要配置智谱GLM的API Key（可免费注册获取），对话费用由智谱收取

**Q: 会不会泄露我的聊天内容？**
A: 绝对不会！所有消息内容仅在本地处理，不上传
